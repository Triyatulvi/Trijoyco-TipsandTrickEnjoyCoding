Selamat Belajar Bahasa Pemrograman Java

Java adalah bahasa pemrograman yang biasa digunakan untuk mengembangkan bagian back-end dari software, aplikasi Android, dan juga website. Java juga dikenal memiliki moto “Write Once, Run Anywhere”. Artinya, Java mampu dijalankan di berbagai platform tanpa perlu disusun ulang menyesuaikan platformnya. Misalnya, berjalan di Android, Linux, Windows, dan lainnya. Hal itu dapat terjadi karena Java memiliki sistem syntax atau kode pemrograman level tinggi. Di mana ketika dijalankan, syntax akan di-compile dengan Java Virtual Machine (JVM) menjadi kode numeric (bytescode) platform. Sehingga aplikasi Java bisa dijalankan di berbagai perangkat.
